# n8n

Open-source, low-code workflow automation platform with AI agent capabilities. Often described as "the power user's Zapier." (Source: [[S — Best AI Tools 2026]])

## Key Features

- AI Agent nodes that enable self-correcting workflows — agents reason through errors and handle edge cases autonomously
- Chain LLMs with operational tools (Slack, HubSpot, databases, APIs) in visual workflows
- Free when self-hosted / $20/mo cloud-hosted
- Open-source model means no data leaves your infrastructure if self-hosted

## Why It Matters

n8n is the leading orchestration platform for technical teams in 2026. Its AI Agent nodes apply the same [[Coding Agent Architecture]] loop pattern — plan, execute, observe, correct — to business workflows rather than code. This makes it a practical bridge between [[Agentic AI]] research and real-world automation.

The open-source, self-hosted option is a major differentiator for teams with data sensitivity requirements, aligning with the broader [[Open-Source AI]] movement.
