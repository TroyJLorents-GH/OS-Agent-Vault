### 🖼️ Drag-and-Drop Dashboard – Picture Drop Flow
1. User **selects an image widget** from a sidebar
2. **Drags it** over the dashboard canvas
3. A **highlighted drop zone** appears on the grid
4. User **releases** → image snaps into place
5. Image can be **resized, repositioned, or replaced**

---

### 🎨 Key Visual UX Elements
- **Drop zone highlight** — border glow or color change when hovering
- **Ghost/shadow preview** — semi-transparent image follows the cursor
- **Grid snapping** — image locks to grid on release
- **Placeholder state** — dashed border + 🖼️ icon before image is dropped
- **Resize handles** — adjust image panel size after dropping
